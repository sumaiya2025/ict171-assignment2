Project Proposal

My cloud project is a photography blog website hosted on an Infrastructure as a Service (IaaS) platform using DigitalOcean. The purpose of this project is to learn how to configure and manage a secure, cloud-based Linux server and use it to host a basic website. The site showcases a few sample photographs along with short descriptions. It represents both a technical deployment task and a personal creative space.

For this project, I deployed an Ubuntu droplet on DigitalOcean and installed the Nginx web server to serve my site content. I also purchased a domain name — sumaiya.website — from Web.com, and pointed its DNS records to my server's IP. To make the site secure and accessible, I installed a free SSL certificate from Let’s Encrypt, enabling HTTPS.

The website is built using static HTML and CSS. It includes sample photos and an overview of the blog concept, fulfilling the minimum content requirements. No backend or database was used, as the purpose was to focus on deploying infrastructure manually and understanding how each component works.

Through this task, I gained hands-on experience in managing Linux via SSH, configuring Nginx, securing a site with SSL, managing DNS records, and hosting a website independently. In future, I may expand the blog to include categories such as nature, architecture, or portraits. This project demonstrates my foundational understanding of server environments and cloud infrastructure, which are key outcomes of this unit.
